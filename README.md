## Assistant API backend

Minimal Node.js + Express + MongoDB API for authentication, memories, interactions, tasks, and context aggregation.

### Installation

1. Clone the repo and install dependencies:
   - `git clone https://github.com/wowthrisha/mongo_db.git`
   - `cd mongo_db`
   - `npm install`

2. Create a `.env` file based on `.env.example` and set your `MONGO_URI`, `JWT_SECRET`, and `PORT` (default `4000`).

3. Start the server:
   - `npm run dev`

The API will be available at `http://localhost:4000` and the simple demo UI at `public-demo.html`.
